# Computational characterization of core gene set in archaea
アーキアのコア遺伝子をドメインレベル〜種レベルまで検出して、分類した研究。日本分子生物学会2019で発表。<br>
スライドより少し更新あり。クソコードの倉庫。<br>
<br>
1. アーキア全体でのオーソログ検索<br>
2. 遺伝子が何%のアーキアにプロットされているかで分類、98%で遺伝子数増加(=コア遺伝子?)<br>
3. 系統情報取得、系統の98%以上(大半は100%)に保存されてる遺伝子を検出<br>
4. 遺伝子を保存されている分類群に応じて分類<br>
5. 集計、COGを用いて機能分類<br>
6. tRNAの保存性(コアではなかったCys, Ser, Lys-tRNA ligase)をアーキアの目レベルで分類 (写真2枚目)<br>
7. ribosome上のコア遺伝子と非コア遺伝子産物ribosomal proteinを可視化<br>
